This is a simple Java console application built using Object-Oriented Programming (OOPS) concepts. It helps to manage insurance claims, including filing claims, updating claim status, viewing claim history, and calculating the settlement amount.It is mainly created as a learning project to understand Java OOPS principles and exception handling.
